# Japanese Cultural Portfolio "Miyabi" (雅)

A deeply immersive, cinematic personal portfolio website celebrating Japanese culture, aesthetics, and technical craftsmanship. This project aims to blend traditional "Wafuu" design with modern web technologies (HTMX, Three.js) to create a "wow" factor experience.

## 🌸 Design Philosophy

- **Cinematic Experience:** The site operates like a film or game, with smooth transitions, atmospheric audio, and dynamic visual effects (falling sakura, fog).
- **Wafuu (Japanese Style):** Usage of traditional colors (Nippon Colors), vertical typography (Tategaki), and textures (Washi paper).
- **Immersive storytelling:** The user explores the portfolio like a journey through a Japanese landscape—starting at the Torii gate, moving through gardens (Projects), and resting at a Ryokan (Contact).

## 🏯 Core Features

1.  **Entry Gate (Landing):** A dramatic 3D/2.5D visual of a Torii gate with falling cherry blossoms.
2.  **The Journey (Navigation):** Smooth, single-page-app feel using HTMX for transitions without full reloads.
3.  **The Garden (Work):** Portfolio items displayed as stones or lanterns in a Zen garden.
4.  **The Scroll (Bio):** A horizontal or vertical scrolling section resembling a traditional EMakimono (handscroll).
5.  **Atmosphere:**
    - Procedural Sakura particle system (Three.js).
    - Ambient soundscapes (optional toggle).
    - "Glassmorphism" with Washi paper textures.

## 🛠 Tech Stack

- **Core:** HTML5, Vanilla JavaScript, CSS3.
- **Interactivity:** HTMX (for seamless swapping of content).
- **3D/Graphics:** Three.js (for particle effects and depth).
- **Styling:** Custom CSS with CSS Variables for themes (Day/Night or Seasonal).
- **Fonts:** Zen Old Mincho / Noto Serif JP for typography.

## 📂 Visual Assets Planned

- **Hero:** Mt. Fuji & Torii Gate at magic hour.
- **Elements:** Katana, Samurai Armor (Dō-maru), Bonsai, Koi Fish.
- **Textures:** Rice paper, Tatami, Gold leaf.
